What is HTML Structure?

The structure of an HTML document defines how its elements are organized. It consists of different sections, such as the head and body, which are required to organize and display content.

Key Components of HTML Structure:

  • DOCTYPE Declaration: This is the first line of an HTML document. It tells the web browser which version of HTML to expect. 
    <!DOCTYPE html>
  • HTML Element: The root element that contains all the HTML content. 
    <html lang="en"></html>
  • Head Element: Contains metadata about the document like the character set, page title, linked stylesheets, etc. 
    <head><meta charset="UTF-8"><meta name="viewport" content="width=device-width, initial-scale=1.0"><title>HTML Structure Example</title></head>
  • Body Element: Contains all the visible content displayed on the web page, such as text, images, and links. 
    <body><h1>This is a Heading</h1><p>This is a paragraph of text.</p></body>

Basic Syntax of HTML Tags

HTML uses tags to structure and display content. Tags are enclosed in angle brackets (< >), and most tags come in pairs: an opening tag and a closing tag. For example:

<p>This is a paragraph.</p>

Some HTML elements, like images, do not require a closing tag. These are called self-closing tags. For example:

<img src="image.jpg" alt="Sample Image">

Common HTML Tags:

  • Headings: HTML provides six levels of headings, from <h1> to <h6>. The <h1> tag represents the most important heading, and <h6> is the least important. 
    <h1>Main Heading</h1><h2>Subheading</h2>
  • Paragraph: The <p> tag is used to define paragraphs. 
    <p>This is a paragraph of text.</p>
  • Anchor (Links): The <a> tag is used to create hyperlinks. 
    <a href="https://example.com">Visit Example</a>
  • Images: The <img> tag is used to embed images. 
    <img src="image.jpg" alt="Sample Image">
  • Lists: HTML supports two types of lists:
    • Unordered List (<ul>): 
      <ul><li>Item 1</li><li>Item 2</li></ul>
    • Ordered List (<ol>): 
      <ol><li>First item</li><li>Second item</li></ol>

Nested HTML Elements

HTML elements can be nested inside other elements. For example, a list of items can be wrapped inside a <div>:

<div class="list-container"><ul><li>Item 1</li><li>Item 2</li></ul></div>

HTML Comments

You can add comments in HTML using <!-- -->. Comments are ignored by the browser and are used for documentation purposes.

<!-- This is a comment -->

Example of HTML Document Structure


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>HTML Structure and Syntax</title>
    </head>
    <body>
        <header>
            <h1>Welcome to HTML Structure and Syntax</h1>
        </header>
        <main>
            <section>
                <h2>Introduction</h2>
                <p>HTML structure consists of several key components, including tags, elements, and attributes.</p>
            </section>
            <section>
                <h2>Syntax Example</h2>
                <p>The syntax of HTML involves using tags to create the structure and content of a webpage.</p>
            </section>
        </main>
        <footer>
            <p>© 2024 HTML Learning</p>
        </footer>
    </body>
</html>
