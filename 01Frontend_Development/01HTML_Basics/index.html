HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure for content on the web and uses "elements" to define different types of content such as text, images, links, videos, and more.

Key Features of HTML

  • Structure: Defines the layout and organization of web pages.
  • Elements and Tags: Uses tags like <p>, <h1>, <div>, etc., to describe content.
  • Attributes: Enhances elements by providing additional information. For example:
<img src="image.jpg" alt="Description of the image">

Hyperlinks: Enables navigation between pages using <a> tags.

Basic HTML Document Structure


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Basics</title>
</head>
<body>
    <h1>Welcome to HTML Basics</h1>
    <p>This is a paragraph.</p>
</body>
</html>

HTML Tags Overview

  • Heading Tags: <h1> to <h6> represent headings.
  • Paragraph Tag: <p> is used for paragraphs.
  • Anchor Tag: <a> creates hyperlinks.
  • Image Tag: <img> embeds images.
  • List Tags: Used to create lists:
    • Ordered List: <ol> for numbered lists.
    • Unordered List: <ul> for bulleted lists.

Attributes in HTML

Attributes provide additional details about elements:

  • Global Attributes:
    • class: Specifies a class for an element.
    • id: Assigns a unique identifier.
    • style: Adds inline styles.
  • Specific Attributes:
    • <img>: src, alt.
    • <a>: href, target.

<a href="https://example.com" target="_blank">Open in New Tab</a>

HTML and Web Browsers

Web browsers like Chrome, Firefox, and Safari interpret HTML code and render it visually for users. They ignore any errors in HTML, making it forgiving but also important to write clean and valid code.

HTML Evolution

HTML has evolved over time, with HTML5 being the latest version offering:

  • New semantic elements like <header>, <footer>, and <article>.
  • Improved multimedia support with <audio> and <video>.
  • Enhanced form controls like <input type="email">.

Why Learn HTML?

  • Foundation of Web Development: It's the starting point for learning web development.
  • Widely Used: All websites rely on HTML for structure.
  • Easy to Learn: It's beginner-friendly and intuitive.

HTML Example


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML Page</title>
</head>
<body>
    <header>
        <h1>Welcome to My Website</h1>
    </header>
    <main>
        <p>This is a basic HTML page created to demonstrate HTML basics.</p>
        <a href="https://example.com">Click here to learn more</a>
        <img src="image.jpg" alt="Sample Image">
    </main>
    <footer>
        <p>© 2024 My Website</p>
    </footer>
</body>
</html>
